This plugin is the child plugin of "Forpublicpost" .Simple install this plugin on your site (sub site).

== Description ==

This plugin is totally depends on "Forpublicpost" plugin (parent). The functionality of this plugin is if you wants to show a parent site post on another site to simple install this plugin and add your cridentials on parent plugin. So your parent site post is showing on your site but in parent site when admin approved the post means if admin want show this post to child site then showing your site other wise not showing. 

== Installation ==

1. Download `childpublicpost.zip` and upload contents to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
